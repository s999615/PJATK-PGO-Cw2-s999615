Projekt Biblioteki
Opis: 
Ten projekt symuluje prosty system biblioteczny. Umożliwia dodawanie książek, wyświetlanie dostępnych książek, ich wypożyczanie i zwracanie.

Funkcje:
- Dodawanie książek do biblioteki
- Wyświetlanie dostępnych książek
- Wypożyczenie książki
- Zwrot książki
- Liczenie dostępnych książek

Klasy:
- Book – przechowuje informacje o książce i jej dostępność
- Reader – przechowuje dane czytelnika i liczbę wypożyczeń
- Library – zarządza książkami i operacjami
- Main – zawiera scenariusz testowy

Przykładowy scenariusz:
1. Dodawanie książek do biblioteki
2. Wyświetlenie dostępnych książek
3. Wypożyczenie książek
4. Wyświetlenie zaktualizowanej listy
5. Zwrot książki
6. Wyświetlenie ostatecznej listt
